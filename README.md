# En Una Nota

Juego musical de fiesta con Spotify y buzzers desde el teléfono.

## Objetivo

Adivina la canción antes que los demás. El primer jugador en llegar a 10 puntos gana la partida.

## Preparación

1. El host abre la pantalla principal del juego.
2. El host presiona **Conectar Spotify** y lee las reglas antes de iniciar.
3. El host elige una playlist de su cuenta.
4. Los jugadores escanean el QR para abrir el buzzer en sus teléfonos.
5. Cada jugador escribe su nombre y presiona **Entrar**.

## Reglas

1. Durante los primeros 10 segundos vale tararear, cantar o decir el nombre de la canción. El nombre del cantante no cuenta como respuesta.
2. Si el host presiona **Continuar**, solo vale decir el nombre exacto de la canción: no se puede cantar, tararear ni hacer melodías.
3. Si respondes mal, pierdes 1 punto y pasa el turno al siguiente jugador en la cola de buzzers.
4. Si nadie sabe la canción, el host puede mostrar la respuesta o pasar a una nueva ronda.

## Cómo Se Juega

1. El host presiona **Reproducir** para iniciar una ronda.
2. Suena un aviso corto y luego se reproduce el fragmento inicial de una canción.
3. Cuando un jugador cree saber la respuesta, presiona **BUZZ**.
4. La música se detiene y el host revisa el orden de respuesta.
5. El primer jugador de la cola responde.
6. El host marca **+1** si acierta o **-1** si falla.
7. Si falla, responde el siguiente jugador de la cola.
8. Si acierta, se muestran los datos de la canción y se limpian los buzzers.

## Controles Del Host

- **Reproducir**: inicia una nueva ronda con una canción aleatoria.
- **Repetir**: vuelve a reproducir el fragmento de la ronda actual.
- **Mostrar**: muestra la respuesta sin sumar puntos.
- **Continuar**: reproduce la canción más allá del fragmento inicial.
- **Pausar**: detiene la canción cuando ya se había presionado Continuar.
- **Buzz de Oro**: reinicia los puntajes a 0; el próximo jugador que acierte gana la partida.
- **Limpiar buzzers**: borra la cola actual de respuestas.
- **Reiniciar juego**: reinicia la partida.

## Controles Desde El Teléfono

Los jugadores solo necesitan el botón **BUZZ**.

Si alguien debe ayudar al host desde el teléfono, puede ingresar la clave de host en la pantalla de entrada. Esa clave activa controles para reproducir rondas, mostrar respuestas, continuar canciones, limpiar buzzers y sumar o restar puntos a quienes hayan buzzeado.

## Cambio De Playlist

Al elegir una nueva playlist, la partida se reinicia y todos los puntajes vuelven a 0. Los jugadores conectados se mantienen en la sala.

## Ganador

Gana quien llegue primero a 10 puntos. En **Buzz de Oro**, gana directamente el primer jugador que acierte.
